# BCA Attendance System

A comprehensive attendance management system for BCA department with 3 years and 3 sections each.

## Features

- 👨‍💼 **Admin Panel**: Add students/staff, view class-wise attendance
- 👨‍🏫 **Staff Panel**: Mark attendance period-wise (6 periods)
- 🎓 **Student View**: View personal attendance with period-wise details
- 📊 **Attendance Calculation**: Period 1 & 4 (3 points), Others (0.75 points)
- 📱 **Responsive Design**: Works on desktop and mobile
- 💾 **Data Storage**: SQLite database

## Tech Stack

- **Backend**: Flask (Python)
- **Database**: SQLite with SQLAlchemy
- **Frontend**: HTML5, CSS3, JavaScript
- **Data Processing**: Pandas, OpenPyXL
